# QuestLog

QuestLog is a small beginner-friendly web app for tracking your video game backlog.

The goal is to keep it simple, clean, and easy to use while still feeling modern. QuestLog lets you log games, organize them by status, save data locally in the browser, and pull in game metadata for a more visual collection-style layout.

## Planned Features

- Add games to your backlog
- Track status like Backlog, Playing, and Finished
- Save data with localStorage
- Pull game metadata and cover art
- Filter and manage your collection in a clean UI

This project is still in progress.
